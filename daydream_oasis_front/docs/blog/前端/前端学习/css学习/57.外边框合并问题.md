```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>外边框合并的问题</title>
</head>

<style>
    * {
    padding:0;
    margin:0;
        }


    .father {
        height:200px;
        background-color:red;
        overflow:hidden;
    }

    .son {
        height:120px;
        background-color:purple;
        margin:50px auto;
        }

</style>

<body>
<!--对于两个嵌套关系的块级元素，如果父级元素没有上内边距及边框，则父元素的上外边距会与子元素的上外边距发生合并，合并后的外边距为两者中的较大者，
即使父级元素的上外边距为0，也会发生-->
<!--解决方案:
1.为父级元素定义一个上内边距或上内边框
2.可以为父级元素添加overflow:hidden;-->
<div class="father">
    <div class="son"></div>
</div>


</body>
</html>
```
