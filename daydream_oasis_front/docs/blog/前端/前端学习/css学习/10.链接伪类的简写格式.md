```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>链接伪类的简写格式</title>
    <style>
        a {
        font-size:16px;
        font-weight:700;
        color:black;
        }
        a:hover{
        font-size:16px;
        font-weight:700;
        color:red;
        }
    </style>
</head>
<body>
<!--在开发中，实际上不需要写那么都的伪类状态,通常加一个hover伪类选择器就行了(鼠标经过时的状态)-->
<a href="#">未知链接</a>
</body>
</html>
```
