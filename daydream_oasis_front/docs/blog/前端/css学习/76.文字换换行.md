```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>文字换行(white-space)</title>
       <style>
        div {
            width:100px;
            height:20px;
            border:1px solid red;
            white-space:nowrap;
            }
    </style>
</head>
<body>
1.解释:white-space设置或检索对象内文本的显示方式，通常我们使用强制一行显示文本
normal:默认处理方式
nowrap:强制在同一行内显示所有文本，直到文本结束或遭遇br标签对文本换行
<div>
    你在干什么呀?
</div>
</body>
</html>
```
