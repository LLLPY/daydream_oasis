<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>外部样式表</title>
    <link href="style1.css" rel="stylesheet">
</head>
<body>
1.解释: <br>
结构用一个html文件写,样式再用另外一个css文件写，然后将两个文件连接起来 <br>
<!--2.连接方法: <br>-->
<!--在head标签中，添加<link href="css文件路径" rel="stylesheet">标签即可-->
