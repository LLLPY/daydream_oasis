```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>交集选择器</title>
    <style>
        .red {
            color:red;
            }
        div.red {        /*交集选择器*/
        font-size:100px;
        }
        span.red {      /*交集选择器*/
        font-size:50px;
            text-shadow:3px 2px 3px rgba(0,0,0,1);
        }
    </style>
</head>
<body>
1.解释: <br>
交集选择器由两个选择器构成，其中一个为标签选择器，第二个为class(类)选择器，两个选择器之间不能有空格 <br><br><br>

2.例: <br>
<div>熊大</div>
<div>熊二</div>
<div class="red">光头强</div>
<span>张三</span> <br>
<span class="red">李四</span> <br>
<span>王五</span> <br>
</body>
</html>
```
