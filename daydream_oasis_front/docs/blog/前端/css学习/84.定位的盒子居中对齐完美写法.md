```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>居中对齐的完美写法</title>
    <style>

    div {
        width:200px;
        height:200px;
        background-color:skyblue;
        position:absolute;
        left:50%;
        top:50%;  /*首先水平，垂直方向各移父级高宽的一般，然后再分别回退自己高宽一半的距离*/
        transform:translate(-50%,-50%); /*translate中使用%不是以父级为准的，而是以自身的宽度和高度为准来进行取值的*/
        ]


    </style>

</head>
<body>


<div></div>


</body>
</html>
```
