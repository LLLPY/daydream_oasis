```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>颜色半透明</title>
    <style>
        p {
        font-size:50px;
        font-weight:900;
        color:rgba(255,0,255,0.3);   /*rgba是设置颜色的方式之一(红色的颜色深度(0~255),绿色我的颜色深度(0~255),蓝色的颜色深度(0~255),透明度(0~1))*/
        }  /*设置为3成透明*/

    </style>
</head>
<body>
<p>css3以后颜色可以设置透明度</p>
</body>
</html>
```
