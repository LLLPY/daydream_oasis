```python
class Cat:
    def __init__(self):
        self.name = "Tom"

    def eat(self):
        print("%s 爱吃鱼"%self.name)

#使用类名（）创建对象的时候，会 自动调用 初始化方法__init__
tom = Cat()
tom.eat()

lazy_cat = Cat()
lazy_cat.eat()
#显然不管是tom还是lazy_cat对象他们的name都被固定为Tom，显然代码还存在缺陷
```
