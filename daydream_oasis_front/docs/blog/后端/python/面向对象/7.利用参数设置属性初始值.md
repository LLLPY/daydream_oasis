```python
class Cat:
    def __init__(self,name):
        self.name = name

    def eat(self):
        print("%s 爱吃鱼"%self.name)

#使用类名（）创建对象的时候，会 自动调用 初始化方法__init__
tom = Cat("Tom")
tom.eat()

lazy_cat = Cat("lazy_cat")
lazy_cat.eat()
```
