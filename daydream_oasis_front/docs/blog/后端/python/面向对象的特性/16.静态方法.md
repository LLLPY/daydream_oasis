```python
class Dog(object):

    #在不需要访问类属性/实例属性,或调用类方法/实例方法的时候，可以定义静态方法

    @staticmethod
    def run():
        print("小狗爱跑.....")


#调用静态方法
#创建静态方法的好处在于：不需要创建实例，就可以对其方法进行调用，通过 类名. 的方式调用

Dog.run()

wangcai = Dog()
wangcai.run()
```
