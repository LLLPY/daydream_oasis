```python
class A:

    def test(self):
        print(" test 方法")

class B:

    def demo(self):
        print(" demo 方法")

#多继承：子类可以同时继承多个父类的方法和属性
class C(A,B):
    pass

c = C()
c.demo()
c.test()
```
