# 如果想判断对象能否被调用，可以使用callable()函数进行判断。
# 例：
print(callable(len))

# python中有7中可调用对象
'''
1.用户定义的函数
    使用def语句或lambda表达式创建
    
2.内置函数
    使用C语言（Cpython）实现的函数，如len。

3.内置方法
    使用C语言实现的方法，如dict.get

4.方法
    在类中定义的函数

5.类
    调用类时会运行类的__new__方法创建一个实例，然后运行__init__方法，初始化实例，最后把实例返回给调用方。

6.类的实例
    如果类定义了__call__方法，那么它的实例可以作为函数调用
    
7.生成器函数
    使用yield关键字的函数或者方法。调用生成器函数返回的是生成器对象。

'''
