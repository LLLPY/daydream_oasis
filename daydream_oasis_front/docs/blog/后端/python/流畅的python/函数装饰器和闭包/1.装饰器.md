```python
# 装饰器是可调用对象，其参数是另一个函数（被装饰的函数）。装饰器可能会处理被装饰的函数，
# 然后把它返回，或者将其替换成另一个函数或者可调用对象


def decorate(func):
    print(func)
    def inner():
        print('run inner...')

    return inner


@decorate
def target(a=1,b=2):
    print('run target...')


if __name__ == '__main__':
    # 可以看到运行的是inner()函数
    target()

    '''
    因此可以看到装饰器的来个特性：
        1.能把被修饰的函数替换成其他函数
        2.装饰器在加载模块时立即执行
    '''

```
