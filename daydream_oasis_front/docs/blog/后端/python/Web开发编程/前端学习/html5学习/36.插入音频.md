1.语法:
<h3><!--
    <audio src="音频地址" autoplay="autoplay" loop="-1" controls></audio>
    src:音频地址
    autoplay:自动播放
    loop:循环次数(-1是无限循环)
    controls:显示控制组件

    <audio controls loop="-1" autoplay="autoplay">
        <source src="Dance Monkey.mp3">
        <source src="Dance Monkey.flac">
        (由于浏览器的兼容问题，为确保音频能正常播放，需要插入不同格式的音频)

    </audio>
     -->
</h3>


<h2>来欣赏一首音乐吧!</h2>

</body>
