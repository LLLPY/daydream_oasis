```python
xiaoming = {"name" : "小明",
            "age" : 18,
            "gender" : "man",
            "heigh" : 1.85}

#迭代遍历字典
#变量k是每一次循环中，获取的键值对的key

for k in xiaoming:
    print("%s - %s"%(k,xiaoming[k]))
```
