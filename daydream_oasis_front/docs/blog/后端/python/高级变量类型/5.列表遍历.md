```python
name_list = ["张三","李四","王五"]

#使用迭代（iteration）遍历
"""
顺序地从列表中获取数据，每一次循环过程中，数据都会保存
在my_name这个变量中，在循环体内部可以访问到当前这一次
获取的数据
"""
for my_name in name_list:
    print("我的名字叫：%s"%my_name)
```
