```python
from functools import reduce

a = [1,2,3,4]
def add(x,y):
    return x+y

def sub(x,y):
    return x-y

#传入的函数必须要有两个参数
#首先将序列的前两个值传入，根据传输函数对参数的操作，返回一个值，并将这个值作为第二次map操作的一个参量，从序列读入第三个
#数作为第二个参数，将这两个参数再传给传入的函数，一次类推

#序列求和
x = reduce(add,a)
print(x)

#序列之差
y = reduce(sub,a)
print(y)

#把序列[1,3,4,3]变成整数1343
def adds(x,y):
    return x*10 + y
a = [1,3,4,3]
sum = reduce(adds,a)
print(sum)






```
