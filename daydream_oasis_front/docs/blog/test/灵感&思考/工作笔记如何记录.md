<blockquote>
<p>工作时，在有笔记要记录时，在本子上，有必要进行分类记录。</p>
</blockquote>
<p>内容类别如下：</p>
<ul>
<li>随手记  “一些不重要的，随便记记”</li>
<li>会议记录  “会议内容”</li>
<li>工作任务</li>
<li>重要</li>
<li>灵感&amp;思考</li>
</ul>
<p>如：</p>
<p>230908/会议记录/周一早会</p>
<ul>
<li>活力指标、汇款指标</li>
<li>如何在wiki 上找到想要的资料。（平时多看下）</li>
</ul>
<p>230908/工作任务/每日计划</p>
<ul>
<li>培训视频看完</li>
<li>开机自启。</li>
</ul>
